Topic 4: Stretching Numbers (Expanded Form)

Chapter Home > Topic 4

What is Expanded Form?

Expanded form is like stretching a number out like a rubber band! It helps us see the real value of each digit inside the number.

Instead of just seeing '345', we see how the '3', '4', and '5' each add up to make the total.

How to Stretch a Number

It's easy! Just follow these steps:

  1. Look at each digit in the number.
  2. Figure out its place value (Ones, Tens, Hundreds, Thousands, etc.).
  3. Multiply the digit by its place value. (Example: If '8' is in the Hundreds place, its value is 8 x 100 = 800).
  4. Write down the value of each digit as an addition sum.

Example: Stretching 82,743

  • 8 is in the Ten Thousands place: 8 x 10,000 = 80,000
  • 2 is in the Thousands place: 2 x 1,000 = 2,000
  • 7 is in the Hundreds place: 7 x 100 = 700
  • 4 is in the Tens place: 4 x 10 = 40
  • 3 is in the Ones place: 3 x 1 = 3

So, the expanded form is: 80,000 + 2,000 + 700 + 40 + 3

Example: Stretching 6,15,496

  • 6 is in the Lakhs place: 6 x 1,00,000 = 6,00,000
  • 1 is in the Ten Thousands place: 1 x 10,000 = 10,000
  • 5 is in the Thousands place: 5 x 1,000 = 5,000
  • 4 is in the Hundreds place: 4 x 100 = 400
  • 9 is in the Tens place: 9 x 10 = 90
  • 6 is in the Ones place: 6 x 1 = 6

So, the expanded form is: 6,00,000 + 10,000 + 5,000 + 400 + 90 + 6

Putting Stretched Numbers Back Together

You can also go backwards! If you see a stretched-out number, you can write the regular number (called the numeral).

Example: Building from 7,00,000 + 5,000 + 40 + 5

  • 7,00,000 means '7' in the Lakhs place.
  • There's no Ten Thousands, so we need a '0' there.
  • 5,000 means '5' in the Thousands place.
  • There's no Hundreds, so we need a '0' there.
  • 40 means '4' in the Tens place.
  • 5 means '5' in the Ones place.

Putting it together (using zeros as placeholders!): 7,05,045

Example: Building from 9,00,000 + 70,000 + 800 + 20

  • 9,00,000 -> '9' in Lakhs place.
  • 70,000 -> '7' in Ten Thousands place.
  • No Thousands -> '0' in Thousands place.
  • 800 -> '8' in Hundreds place.
  • 20 -> '2' in Tens place.
  • No Ones -> '0' in Ones place.

Putting it together: 9,70,820

Place Value is Key!

See? Expanded form is all about understanding place value! Each part of the stretched-out number shows exactly what a digit is worth because of where it lives in the number.
