Let's Be Number Architects!

Now that you know about place value, comparing, and ordering, let's build our own numbers using rules!

Building with Given Digits

Sometimes, you'll get a set of digits (like number cards) and need to make a number.

Example: Use the digits 8, 2, 5 to make the smallest number (using each digit once).

  • To make the smallest number, put the smallest digit in the highest place value (leftmost).
  • Smallest digit is 2. Put it first.
  • Next smallest is 5. Put it second.
  • Largest is 8. Put it last.

The smallest number is 258.

What about the largest number using 8, 2, 5? (Hint: Put the largest digit first!)

Building with Place Value Clues

Other times, you get clues about which digit goes in which place value house.

Example: Build a number with:

  • 5 at Lakhs place
  • 8 at Ten Thousands place
  • (No Thousands mentioned!)
  • 3 at Hundreds place
  • 1 at Tens place
  • 4 at Ones place

Remember Zero the Hero! If a place isn't mentioned, put a zero there.

The number is: 5,80,314

Building with Special Rules (Smallest/Largest/Even/Odd)

Sometimes there are extra rules!

  • Smallest/Largest: Remember to put smaller digits first for the smallest number, and larger digits first for the largest number. Watch out if zero is one of the digits â€“ it usually can't go in the very first spot (unless it's the only digit!).
  • Even/Odd: An even number ends in 0, 2, 4, 6, or 8. An odd number ends in 1, 3, 5, 7, or 9. Make sure the digit in the Ones place follows the rule!

Example: Smallest 6-digit number with 8 at Hundreds place.

  • Smallest 6-digit number starts with 1 (can't be 0).
  • We need 8 in the Hundreds place.
  • Fill the other places with the smallest possible digit (0).

The number is: 1,00,800

Example: Largest 6-digit even number starting with 5 (using digits 0, 1, 2, 3, 4, 5, 6, 8, 9 once each - simplified example).

Let's use digits 5, 9, 8, 2, 1, 0 to make the largest 6-digit even number starting with 5.

  • Starts with 5: 5 _ _ _ _ _
  • To be largest, use the next biggest digits: 9, 8, 2, 1, 0.
  • To be even, it must end in 0, 2, or 8. Let's use the smallest even digit (0) at the end to keep the other places large.
  • Fill the middle with the remaining largest digits (9, 8, 2, 1) in descending order.

The number is: 5,98,210

Finding the Next Number (Successor)

Sometimes, after building a number, you might be asked for its successor. That's just the number that comes right after it (add 1!).

Example: The successor of 258 is 259.
